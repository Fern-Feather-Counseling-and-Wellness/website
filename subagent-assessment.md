## Sub-Agent Options

### 1. Content Agent
**Purpose:** Ongoing TikTok and social media content creation

**Would handle:**
- Draft TikTok scripts based on trending formats
- Create carousel posts for Instagram
- Write email newsletter content
- Adapt existing content for different platforms

**Pros:**
- Content needs are ongoing and repetitive
- Can work within established voice/tone guidelines
- Does not require web search for basic content adaptation

**Cons:**
- Without web search, can't research trending topics
- Would need Nicole to feed trending ideas/topics
- May create content that feels formulaic without fresh input

**Verdict:** Useful but limited without web search fix

### 3. Admin/Systems Agent
**Purpose:** Handle operational tasks once practice launches

**Would handle:**
