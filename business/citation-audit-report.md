## 📋 APA CITATION TEMPLATE

### In-Text Citations

**Parenthetical (at end of sentence):**
```
Research shows that somatic therapy reduces anxiety symptoms (Smith, 2024).
```

**Narrative (author name in sentence):**
```
According to Smith (2024), somatic therapy reduces anxiety symptoms.
```

**Multiple authors:**
```
(Smith & Jones, 2024) — 2 authors
(Smith et al., 2024) — 3+ authors
```

**Multiple citations (alphabetical order):**
```
Studies show consistent results (Brown, 2023; Jones, 2024; Smith, 2025).
```

**Direct quote:**
```
As Smith (2024) noted, "The body holds trauma in ways that talk therapy cannot access" (p. 45).
```

---

### Reference List Format

**Journal Article:**
```
Author, A. A., & Author, B. B. (Year). Title of article: Subtitle if needed. *Title of Journal, volume*(issue), page range. https://doi.org/xx.xxx/xxxx

Example:
Grossman, P., Taylor, A. G., & Van Bockstaele, E. (2026). Critical examination of polyvagal theory: Methodological and conceptual issues. *Clinical Neuropsychiatry, 23*(1), 15-32. https://doi.org/10.36131/cn20260123
```

**Book:**
```
Author, A. A. (Year). *Title of work: Capital letter also for subtitle*. Publisher.

Example:
Porges, S. W. (2011). *The polyvagal theory: Neurophysiological foundations of emotions, attachment, communication, and self-regulation*. W. W. Norton & Company.
```

**Book Chapter:**
```
Author, A. A. (Year). Title of chapter. In E. E. Editor & F. F. Editor (Eds.), *Title of book* (pp. xx-xx). Publisher.

Example:
Siegel, D. J. (2020). Interpersonal neurobiology and the brain. In S. W. Porges & D. Dana (Eds.), *Clinical applications of the polyvagal theory: The emergence of polyvagal-informed therapies* (pp. 45-62). W. W. Norton & Company.
```

**Website:**
```
Author, A. A. or Organization Name. (Year, Month Day). Title of page. Site Name. URL

Example:
National Institute of Mental Health. (2023, January). Anxiety disorders. https://www.nimh.nih.gov/health/topics/anxiety-disorders
```

**Government Report:**
```
Organization Name. (Year). *Title of report* (Report No. XXX). Publisher. URL

Example:
Centers for Disease Control and Prevention. (2023). Mental health surveillance among children: United States, 2013-2019. https://www.cdc.gov/mmwr/volumes/72/ss/ss7205a1.htm
```

## 🔄 ONGOING CITATION WORKFLOW

### Before Publishing Any Content:

1. **Draft content** → Write original material
2. **Mark claims** → Highlight every statistic, research finding, clinical assertion
3. **Find sources** → Locate peer-reviewed sources for each claim
4. **Add citations** → Insert in-text citations (Author, Year)
5. **Create reference list** → Full APA format at end
6. **Review** → Check format, alphabetical order, hanging indents
7. **Publish** → Only after reference list is complete

### When Creating New Products:

1. **Research first** → Find 10-15 core sources before writing
2. **Build bibliography** → Create reference list as you go
3. **Cite as you write** → Don't wait until end
4. **Save sources** → Keep PDFs in `research/sources/` folder
5. **Document** → Note which sources support which claims

## ⚠️ PLAGIARISM RED FLAGS TO AVOID

❌ **Direct copying** — Never copy >3 words in a row without quotes  
❌ **Paraphrasing without citation** — Even rewritten ideas need attribution  
❌ **Missing references** — Every factual claim needs a source  
❌ **Old citations** — Check research is current (prefer 2015-2026)  
❌ **Unverified statistics** — "Studies show..." without citation is unacceptable  

✅ **Safe practices:**
- Quote directly with proper attribution (rare)
- Paraphrase extensively with citation (common)
- Synthesize multiple sources into original insight (best)
- Create unique exercises/tools based on research (ideal)
