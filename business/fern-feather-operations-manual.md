## Physical Space Operations

### Office Hours
- **Standard:** Monday-Friday, 9am-7pm EST
- **Weekends:** Limited availability for workshops/groups
- **Holidays:** Closed on major holidays with 4 weeks notice

### Room Usage Protocol
1. **Therapist Offices:** Assigned to primary practitioner
2. **Group Room:** Shared space, booking via shared calendar
3. **Yoga/Workshop Space:** Requires 24-hour setup notice
4. **Common Areas:** Maintained daily by cleaning service

### Safety Systems
- **Flic Smart Buttons:** Panic alerts in each room
- **Silent Beacon Wearables:** Optional for practitioners
- **Emergency Protocols:** Posted in each room
- **After-hours:** Security system armed when empty

### Maintenance
- **Weekly:** Deep clean by service
- **Daily:** Practitioner tidying
- **Monthly:** Equipment check (sound machines, chairs, etc.)
- **Quarterly:** Professional deep clean

## Client Operations

### Intake Process
1. **Initial Contact:** Website form → auto-response within 24 hours
2. **Consultation Call:** 15-minute free consultation
3. **Paperwork:** Digital intake forms via SimplePractice
4. **First Session:** Orientation to space and process

### Documentation Requirements
- **Intake Form:** Completed before first session
- **Informed Consent:** Signed digitally
- **Financial Agreement:** Clear payment terms
- **Privacy Practices:** HIPAA-compliant acknowledgment
- **Release of Information:** As needed

### Session Management
- **Length:** 50-minute clinical hour (45 therapy + 5 notes)
- **Frequency:** Weekly recommended, biweekly optional
- **Cancellation:** 24-hour policy, fee for late cancels
- **No-shows:** Full fee charged, follow-up protocol

### Communication Protocol
- **Primary:** Secure messaging via SimplePractice
- **Secondary:** Encrypted email
- **Emergency:** Phone with clear boundaries
- **Response Time:** 24-48 hours for non-urgent messages

## Financial Operations

### Fee Structure
- **Individual Therapy:** $150-200/session (sliding scale available)
- **Couples Therapy:** $200-250/session
- **Group Therapy:** $60-80/session
- **Workshops:** $75-150 depending on length
- **Supervision:** $100-150/hour

### Payment Processing
- **Platform:** SimplePractice integrated payments
- **Methods:** Credit/debit, HSA/FSA
- **Insurance:** Private pay only (no insurance panels)
- **Sliding Scale:** 2 slots per practitioner, documented criteria

### Billing Cycle
- **Payment:** At time of service via card on file
- **Receipts:** Automated after payment
- **Statements:** Monthly upon request
- **Late payments:** 3-strike policy with clear communication

### Financial Assistance
- **Scholarship Fund:** 5% of revenue allocated
- **Referral Program:** Discount for referred clients
- **Package Discounts:** 10% for prepaid 4+ sessions
- **Community Rate:** Limited slots at reduced rate

## Emergency Procedures

### Medical Emergency
1. **Assess situation:** Determine need for 911
2. **Call 911 if needed:** Provide clear address and situation
3. **First aid:** Apply if trained and safe
4. **Document:** Incident report within 24 hours
5. **Follow-up:** With client and support system

### Mental Health Crisis
1. **Assess risk:** Suicide/homicide assessment
2. **Safety plan:** Implement if available
3. **Emergency contacts:** Notify if appropriate
4. **Higher level of care:** Facilitate referral if needed
5. **Documentation:** Detailed note, consultation sought

### Facility Emergency
1. **Evacuate if needed:** Fire, gas leak, etc.
2. **Emergency contacts:** Call building management
3. **Client communication:** Reschedule affected sessions
4. **Repairs:** Coordinate with management
5. **Business continuity:** Remote sessions if possible

### Practitioner Emergency
1. **Coverage plan:** Backup practitioner available
2. **Client notification:** Scripted message for assistant
3. **Session management:** Reschedule or temporary coverage
4. **Return plan:** Gradual return to full caseload
