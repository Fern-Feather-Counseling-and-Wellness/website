## Generation Instructions

### To Generate These Images:

**Option A: Using OpenAI Image Gen Skill (if API key available)**
```bash
python3 /usr/local/lib/node_modules/openclaw/skills/openai-image-gen/scripts/gen.py \
  --prompt "[paste prompt here]" \
  --model dall-e-3 \
  --quality hd \
  --size 1024x1024 \
  --style natural
```

**Option B: Direct via OpenAI Playground**
- Go to https://platform.openai.com/playground/images
- Paste prompts one at a time
- Select DALL-E 3, HD quality, Natural style
- Download and save

**Option C: Canva's AI Image Generator**
- Use Canva's "Magic Media" or "Text to Image"
- Paste prompts
- Select "Dreamy" or "Watercolor" style
- Generate and refine

## Color Reference for Prompts

When specifying colors in prompts, use descriptive language:
- "Sage green" = soft, muted green
- "Warm sand" = beige/tan
- "Soft cream" = off-white
- "Terracotta" = warm earthy orange-brown
- "Deep forest" = dark rich green
