## 📝 To View with Full Styling

The pages are built with external CSS, so viewing on GitHub shows the HTML structure but not the styled design.

**To see the actual styled website:**

### Option 1: Download and Open Locally
1. Download the `website/` folder from GitHub
2. Open any `.html` file in your browser
3. All pages link to each other locally

### Option 2: Deploy (Recommended)
1. Sign up for [Netlify](https://www.netlify.com) (free)
2. Drag & drop the `website/` folder
3. Instant live site with custom domain support

### Option 3: Use a Preview Service
Try one of these to view the styled pages:
- https://raw.githack.com/ (paste GitHub raw URLs)
- https://htmlpreview.github.io/ (paste GitHub file URLs)
